# ACE Mobile Wash & Wax Website

Static, production-ready website for ACE, a mobile wash and wax service in West Bradenton, FL.

## Public Pages

- `index.html` — Home page
- `about.html` — About page
- `pricing.html` — Pricing page

`gallery.html` remains in the project but is unlinked and marked `noindex`.

## Deployment

This project is ready for GitHub and Cloudflare Pages.

1. Push the full folder to a GitHub repository.
2. In Cloudflare Pages, connect the repository.
3. Use no build command.
4. Set the output directory to `/`.
